Sentiment Analysis Through Advanced NLP and Deep Learning

It is critical to understand public opinions toward COVID-19 vaccines in a worldwide epidemic and immunisation efforts. A machine learning model was developed to assess whether the sentiment is positive, neutral, or negative. The project provided an understanding of public attitudes toward the COVID-19 vaccines. Pre-trained models were fine-tunned and an interactive Streamlit app was designed to offer a real-time sentiment analysis, contributing to the broader dialogue on public health communication and vaccine acceptance. Project tools were Python, Streamlit, Hugging Face pre-trained models and Docker.
